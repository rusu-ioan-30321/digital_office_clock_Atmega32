## Overview
This project implements a digital office clock using the ATmega32 microcontroller and the EasyAVR v7 development board.

## Features
- Real-time clock
- Alarm functionality
- Timer mode
- Integrated thermometer

## Hardware
- ATmega32
- EasyAVR v7 development board
- 7-segment display
- Temperature sensor

## Software
- Language: C
- IDE: AVR Studio / Atmel Studio
- Microcontroller frequency: 8 MHz

## What I Learned
- Register configuration
- Timer configuration and interrupts
- ADC usage
